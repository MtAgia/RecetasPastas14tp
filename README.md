 ## Cómo clonar el repositorio:
```
git clone 
```
Ejecuta el siguiente comando para instalar las dependencias del proyecto:

```
npm install
```
## Ejecutar la aplicación
```
npm run dev
```
 En otra terminal ejecutar:
```
json-server --watch db.json --port 3004
```
